📦 Case Study: Otimização de Almoxarifado (Excel & Power BI)

Esta é a página de apresentação de um estudo de caso focado na transformação de dados de estoque em decisões operacionais estratégicas. O projeto demonstra como o uso combinado de Excel e Power BI pode diagnosticar falhas e otimizar o controle de almoxarifado.
🚀 Sobre o Projeto
O objetivo deste repositório é hospedar o código-fonte da landing page que detalha a solução desenvolvida para um problema real de gestão de suprimentos. A página foi desenhada com uma estética minimalista e profissional, focada em legibilidade e experiência do usuário (UX).
🛠️ Tecnologias Utilizadas no Script
HTML5: Estruturação semântica do conteúdo.
CSS3 (Moderno): Uso de Variáveis (Custom Properties), Flexbox, Grid e animações de fade-up.
Typography: Integração com Google Fonts (Cormorant Garamond e DM Mono) para um visual sofisticado.
Design Responsivo: Adaptável para dispositivos móveis e desktops.
📋 Funcionalidades da Página
Seção Hero: Título impactante com subtítulo explicativo e badges de status.
Navegação Fixa: Menu superior com efeito de desfoque (glassmorphism) para fácil acesso.
Banners Informativos: Espaços dedicados a notas de projeto e metas alcançadas.
Botões de Ação: Links rápidos para visualização de relatórios, dashboards ou contato.
🎨 Identidade Visual
A página utiliza uma paleta de cores "Earth Tones" (tons de terra) para transmitir seriedade e clareza:
Accent: Terracota (#b5451b)
Background: Creme/Sands (#f5ede4)
Typography: Dark Brown (#2c1a12)
